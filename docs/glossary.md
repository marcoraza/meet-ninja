# Glossary

- Transcript: conteúdo bruto ou semibruto gerado por uma reunião.
- Receipt: registro auditável de como uma reunião foi processada.
- Vault: base local de conhecimento do Marco, normalmente em Markdown.
- Dry-run: execução que calcula e mostra a ação sem escrever fora do repositório.
- Capability: ação pequena e explícita que o agente pode usar.
- Source hash: hash do transcrito original usado para deduplicação e auditoria.
