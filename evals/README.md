# Evals

Avaliações de qualidade do agente.

Usar para medir:

- extração correta de decisões;
- extração correta de tarefas;
- confiança adequada;
- roteamento correto para vault;
- payload GitHub bem formado;
- rejeição segura quando faltam dados.

Evals devem usar fixtures artificiais ou anonimizadas.
