# Prompts

Prompts versionados do agente ficam aqui.

Regras:

- Prompts devem pedir evidência, não só resumo.
- Baixa confiança deve aparecer na saída.
- Prompts não podem instruir escrita externa.
- Prompts devem produzir JSON validável quando usados no pipeline.

Prompts longos devem ter teste ou eval correspondente.
